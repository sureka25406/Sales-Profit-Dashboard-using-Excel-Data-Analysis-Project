📊Sales & Profit Dashboard – Excel Data Analysis Project
📌 Project Overview

This project focuses on analyzing a retail sales dataset using Microsoft Excel. The data was imported from a single source and transformed into meaningful business insights through data preparation, Pivot Table analysis, and interactive dashboard creation.

📂 Dataset

The dataset includes the following fields:

Order Date

Customer Name

State

Category

Sub-Category

Product Name

Sales

Quantity

Profit

The entire analysis was performed using this single Excel dataset.

🧹 Data Preparation

Cleaned and structured the dataset

Verified column headers and corrected data types

Formatted numeric and date columns properly

Converted the dataset into an Excel Table

Applied Conditional Formatting to highlight profit and loss

📊 Pivot Table Analysis

Created multiple Pivot Tables to summarize:

Customer count by year

Sales by state

Sales by month

Top 5 customers by profit

These summaries helped identify sales trends, regional performance, and high-value customers.

📈 Dashboard Features

The interactive Excel Dashboard includes:

KPI Cards (Total Sales & Total Profit)

Sales by Category chart

Profit trend by year

Monthly sales analysis

Geographic sales distribution

Slicers for Year, Month, and Category

The dashboard enables dynamic filtering and provides clear business insights.

🚀 Skills Demonstrated

Data Cleaning & Preparation

Excel Table Management

Conditional Formatting

Pivot Table Analysis

KPI Tracking

Dashboard Design

🎯 Outcome

This project demonstrates how a single-source dataset can be transformed into a structured table, summarized with Pivot Tables, and visualized through an interactive dashboard to support data-driven decision-making.
